## 브루트 포스 (Brute Force)

### 개념

- 가능한 **모든 경우의 수를 전부 시도**해보는 완전 탐색(Exhaustive Search) 방식.
- 효율성은 낮지만, **정답을 반드시 찾을 수 있는 확실한 방법**이다.
- 복잡한 문제를 처음 구조화하거나, 다른 알고리즘을 적용하기 전 **기본 접근법**으로 자주 사용된다.

---

### 예시: 두 수의 합 찾기

> 배열에서 두 수를 골라 합이 `target`이 되는 조합이 존재하는지 찾는 예제.

```java
public class BruteForceExample {
    public static void main(String[] args) {
        int[] nums = {2, 7, 11, 15};
        int target = 9;

        boolean found = false;
        for (int i = 0; i < nums.length; i++) {
            for (int j = i + 1; j < nums.length; j++) {
                if (nums[i] + nums[j] == target) {
                    System.out.printf("찾음: %d + %d = %d%n", nums[i], nums[j], target);
                    found = true;
                }
            }
        }

        if (!found) {
            System.out.println("조건을 만족하는 조합이 없습니다.");
        }
    }
}
```

**출력**

```
찾음: 2 + 7 = 9
```

---

### 알고리즘 문제 풀이 관점에서의 중요 포인트

| 구분            | 내용                                                          |
| ------------- | ----------------------------------------------------------- |
| **기본 접근법**    | 문제를 처음 접했을 때 구조를 이해하기 위해 가장 먼저 시도할 수 있는 방법                  |
| **정답 보장**     | 가능한 모든 경우를 탐색하므로 정답이 있으면 반드시 찾아냄                            |
| **시간 복잡도 고려** | 입력 제한이 작을 때만 실용적 (예: `N ≤ 10` 또는 `N ≤ 20`)                  |
| **최적화 전 단계**  | 백트래킹, DFS/BFS, DP, 비트마스크 등으로 확장하기 전에 기준점(ground truth)으로 사용 |


**시간 복잡도:** `O(N²)`
입력 크기가 커질수록 급격히 느려진다.

---

### 주 활용 사례

| 유형                   | 예시                         | 특징             |
| -------------------- | -------------------------- | -------------- |
| **조합/순열 탐색**         | N개의 원소에서 가능한 모든 조합, 순열 생성  | 중첩 루프 또는 재귀 사용 |
| **부분집합 탐색 (Subset)** | 모든 부분집합 탐색                 | 비트마스크로 최적화 가능  |
| **문자열 완전 탐색**        | 모든 문자열 조합 테스트 (예: 비밀번호 추측) | 시간 복잡도 큼       |
| **그래프 탐색 기초**        | 모든 경로 탐색 (DFS/BFS 기반)      | 백트래킹의 기초       |
| **브루트 포스 + 가지치기**    | 불필요한 경우는 조기 중단             | 효율성 향상         |

---

### 한 줄 정리

> 브루트 포스는 **“정답을 보장하지만 느린 방법”** 이며,
> 문제 구조를 파악하고 더 나은 알고리즘으로 발전시키기 위한 **기초 단계**이다.
